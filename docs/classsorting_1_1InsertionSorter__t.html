Detailed Description

template<typename T>
class sorting::InsertionSorter_t< T >

Sorter that implements insertion sort.

Insertion sort is a popular sorting algorithm which runs in O(N^2) time where N represents the length of the array. The idea behind insertion sort is to iterate over the input array while maintaining the invariant that all the elements seen up to a specific iteration are in sorted order. At every iteration, we "insert" the element being considered into the correct position in the already sorted part of the array. Please refer to any text on algorithms (preferably Introduction to Algorithms by Cormen et al) for complete details.

This class extends from the Sorter abstract class and implements the sort() pure function.
