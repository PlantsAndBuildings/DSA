Detailed Description

template<typename T>
class sorting::Sorter_t< T >

Abstract base class for various sorting implementations.

Various concrete sorter implementations such as BubbleSorter and MergeSorter implement the method in this class. A SorterFactory is used to generate objects of this type. A sorter requires (as template argument) a data type - indicating the type of data it is expected to sort. For example, a MergeSorter<int> will be used to sort a vector of ints via the Merge Sort algorithm
