Detailed Description

template<typename T>
class sorting::BubbleSorter_t< T >

Sorter that implements bubble sort.

Bubble sort is a popular sorting algorithm which runs in O(N^2) time where N represents the length of the array. The idea behind bubble sort is to iterate over an array while comparing adjacent elements and swapping them if they are in the wrong order. The array must be iterated over N times for it to be completely sorted. Please refer to any text on algorithms (preferably Introduction to Algorithms by Cormen et al) for complete details.

This class extends from the Sorter abstract class and implements the sort() pure function.
