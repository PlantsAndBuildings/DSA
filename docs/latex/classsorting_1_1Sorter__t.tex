\subsubsection*{template$<$typename T$>$\newline
class sorting\+::\+Sorter\+\_\+t$<$ T $>$}

Abstract base class for various sorting implementations. 

Various concrete sorter implementations such as \mbox{\hyperlink{classsorting_1_1BubbleSorter__t}{Bubble\+Sorter}} and \mbox{\hyperlink{classsorting_1_1MergeSorter__t}{Merge\+Sorter}} implement the method in this class. A \mbox{\hyperlink{classsorting_1_1SorterFactory}{Sorter\+Factory}} is used to generate objects of this type. A sorter requires (as template argument) a data type -\/ indicating the type of data it is expected to sort. For example, a \mbox{\hyperlink{classsorting_1_1MergeSorter__t}{Merge\+Sorter$<$int$>$}} will be used to sort a vector of {\ttfamily int}s via the Merge Sort algorithm
